# audio_classification_scc5920

Repository for code for the SCC5920 (Unstructured Audio Data) discipline's final project. Here are the codes for feature extraction and setup/training of SVMs and CNN using the VGGish architecture.

## Requirements:

- [ESC-50](https://github.com/karolpiczak/ESC-50)
- [Librosa](https://pypi.org/project/librosa/)
- [Matplotlib](https://pypi.org/project/matplotlib/)
- [NumPy](https://pypi.org/project/numpy/)
- [Pandas](https://pypi.org/project/pandas/)
- [PyTorch](https://pytorch.org/)
- [Resampy](https://pypi.org/project/resampy/)
- [Scikit-learn](https://pypi.org/project/scikit-learn/)
- [SoundFile](https://pypi.org/project/SoundFile/)
- [torch_audioset](https://github.com/w-hc/torch_audioset)
- [tqdm](https://pypi.org/project/tqdm/)
